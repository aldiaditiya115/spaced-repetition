JSONBin.io API Key
Cara dapat X-Master-Key gratis:
1. Buka jsonbin.io → daftar gratis
2. Login → klik API KEYS di sidebar kiri
3. Copy key di bagian X-MASTER-KEY (bukan X-Access-Key!)
4. Paste di kolom di atas → klik Lanjut
